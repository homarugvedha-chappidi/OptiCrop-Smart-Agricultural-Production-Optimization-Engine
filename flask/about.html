About OptiCrop

A machine learning-powered platform that helps farmers identify the most suitable crop based on soil nutrients and environmental conditions.

🌱 Machine Learning

Uses a Logistic Regression model trained on agricultural data to recommend the most suitable crop.

📊 Input Parameters

Recommendations are generated using Nitrogen, Phosphorus, Potassium, Temperature, Humidity, pH and Rainfall values.

🎯 Goal

Support smart farming by helping users make informed crop decisions through data-driven recommendations.
